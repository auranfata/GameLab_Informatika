# 🎮 Game Lab – Teknik Informatika

Game Lab adalah sebuah **project eksplorasi dan pembelajaran** yang dikembangkan oleh divisi/komunitas **Teknik Informatika** sebagai media edukatif, interaktif, dan menyenangkan untuk memperkenalkan logika pemrograman, dasar pengembangan game, serta pemanfaatan teknologi web modern.

Project ini ditujukan untuk **masyarakat umum**, pelajar, dan mahasiswa sebagai sarana belajar sambil bermain (*learning by playing*).

---

## ✨ Tujuan Project

- Menjadi media pembelajaran interaktif berbasis web.
- Mengenalkan konsep dasar logika, algoritma, dan *game development*.
- Menyediakan mini games ringan yang mudah dimainkan.
- Menjadi wadah eksplorasi dan pengembangan kreativitas mahasiswa Teknik Informatika.
- Membangun portofolio project berbasis teknologi web dengan standar industri.

---

## 🕹️ Daftar Game

### 1. Tic Tac Toe
Game klasik dua pemain (Player vs Player) yang dibangun dengan arsitektur *Object-Oriented Programming* (OOP).
- **Fitur:** Indikator giliran pemain, deteksi menang/seri, *highlight* visual saat menang, dan efek suara interaktif.

### 2. Bounce Ball
Game *arcade* pantulan bola penghancur bata (Brick Breaker) berbasis Canvas API.
- **Fitur:** Deteksi tabrakan dinamis (*collision detection*), sistem skor, nyawa pemain (*lives*), bata pelindung (tidak bisa hancur), dan kontrol keyboard (kiri/kanan/spasi).

### 3. Memory Card Game
Game asah otak untuk mencocokkan pasangan kartu dengan batas waktu.
- **Fitur:** *Grid layout* responsif, pengacakan kartu (*shuffle*), penghitung waktu mundur (*timer*), sistem skor, dan efek transisi/suara saat kartu dibalik.

### 4. Quiz Informatika
Mini kuis edukatif untuk menguji pengetahuan dasar seputar teknologi dan Teknik Informatika.
- **Fitur:** Pertanyaan pilihan ganda interaktif, validasi jawaban langsung (warna hijau/merah), dan pesan motivasi di akhir game berdasarkan skor rata-rata.

## 🛠️ Teknologi yang Digunakan

- **HTML5 & CSS3** (Global Styling & CSS Variables)
- **JavaScript (Vanilla JS - ES6 Classes)** untuk logika game yang modular
- **Bootstrap 5** (Layout & UI Components)
- **Canvas API** (Rendering visual 2D pada Bounce Ball)
- **FontAwesome** (Ikon UI)
- **Google Fonts** (Poppins)
